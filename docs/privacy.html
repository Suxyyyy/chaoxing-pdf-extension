Privacy Policy

Extension: Chaoxing PDF Link Extractor
Effective date: April 21, 2026

This Privacy Policy explains how Chaoxing PDF Link Extractor ("the Extension") handles information when you use it.

1. What the Extension Does

The Extension helps users extract PDF links from supported Chaoxing courseware pages. It also allows users to configure a custom keyboard shortcut for triggering the extraction action.

2. Information the Extension Accesses

In order to work properly, the Extension may access the following information within supported Chaoxing webpages:

  • Page URLs and embedded frame URLs
  • Courseware-related identifiers such as object IDs
  • Page elements and script content necessary to detect the current courseware resource

The Extension also stores your shortcut preference using the browser's extension storage.

3. How Information Is Used

The accessed information is used only for the following purposes:

  • Detecting the current Chaoxing courseware resource
  • Requesting the corresponding PDF link from Chaoxing
  • Displaying and copying the PDF link for the user
  • Saving the user's shortcut preference

4. Data Storage

The Extension stores only limited local settings, such as your chosen keyboard shortcut, using the browser's extension storage mechanism.

The Extension does not maintain its own external database for user data.

5. Data Sharing and Transmission

The Extension does not sell, rent, or share your personal information with third parties.

The Extension does not transmit your data to any developer-owned server. It only sends requests directly to Chaoxing-related endpoints as required to retrieve the PDF link for the current resource.

6. Personal Information

The Extension is not designed to collect personally identifiable information for analytics, advertising, profiling, or resale.

However, because the Extension operates on logged-in Chaoxing pages, it may access page content or authenticated requests necessary to perform its core functionality.

7. Security

The Extension is designed to minimize data handling and only use the permissions required for its functionality. No guarantee of absolute security can be made, but reasonable efforts are taken to avoid unnecessary data collection.

8. Children's Privacy

The Extension is not specifically directed to children under 13.

9. Changes to This Policy

This Privacy Policy may be updated from time to time. Any changes will be reflected on this page with an updated effective date.
